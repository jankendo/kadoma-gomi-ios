# Search Quality

## 1. 改善方針

検索は「正しいカテゴリを素早く見つける」ことを優先する。完全な自然言語検索ではなく、日常語、表記ゆれ、カテゴリ名、略語、部分一致に強い軽量検索として実装する。

## 2. 対応した表記ゆれ

- ひらがな/カタカナ: `ぺっとぼとる` -> `ペットボトル`
- 半角/全角: `PET`、全角英数、半角カナを同系統に正規化
- 略語: `プラ`
- 類義語: `ダンボール` / `段ボール`
- 呼び方違い: `缶` / `カン`、`びん` / `ビン` / `瓶`
- 部分一致: `ボトル`

## 3. スコアリング

品目名、キーワード、カテゴリ名、カテゴリ短縮名、カテゴリの出し方文を使ってスコアリングする。品目名完全一致を最優先し、カテゴリ短縮名は略語検索で上位に出るようにした。

## 4. 検証ケース

`Scripts/run_quality_checks.py` で以下を確認する。

- `ペットボトル`
- `ぺっとぼとる`
- `PET`
- `ダンボール`
- `段ボール`
- `缶`
- `カン`
- `びん`
- `ビン`
- `プラ`

## 5. 残課題

実利用ログを収集していないため、ランキングは初期辞書ベース。将来は最近検索、検索失敗語、公式LINE相当の語彙追加で強化する。
