## 3. 反映した分別情報

- 普通ごみ: 生ごみ、紙くず、紙おむつ、30cm以下のプラスチック製品など。生ごみは水分を切る。
- プラスチック製容器包装: 食品トレイ、卵パック、レジ袋、ボトル類など。中身を出し、軽く洗い、二重袋にしない。
- びん・缶類: 飲食料用の空きびん・空き缶、スプレー缶、簡易ガスボンベなど。スプレー缶類は使い切り、袋を分けてキケン表示。
- 古紙・古布: 古紙は種類ごとに十字にしばる。古着・タオルは45L以下の袋。
- 小型ごみ: 30cm以下の金物、陶磁器、ガラス、蛍光管、電池、ライターなど。割れ物等は包んでキケン表示。
- ペットボトル: PETマーク付きの飲料・調味料ボトルなど。中身を出し、キャップ・ラベルを外し、軽くすすぐ。
- 粗大ごみ: 30cmを超える耐久消費財など。電話予約、処理券、収集日確認が必要。
- 市で収集できないもの/注意品目: 家電リサイクル対象品、PC、事業系ごみ、危険物などは通常収集に出さない。
