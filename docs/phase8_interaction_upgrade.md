# Phase 8 Interaction Upgrade

## 1. 目的

第8フェーズでは、前フェーズまでの門真市限定データ基盤、A-F地区対応、大倉町1-20プリセット、検索辞書、分別データ、ライトモード専用、シンプルUIを維持したまま、毎日の操作で必要になる「動かせる・開ける・更新できる・テストできる」を追加した。

## 2. 実装した操作

- カレンダー月表示を左右スワイプで前月・翌月へ移動できるようにした。
- ホームの今週リストに表示されるごみ種別から、該当分別カテゴリ詳細へ遷移できるようにした。
- カレンダーセル内のごみ種別ラベルから、該当分別カテゴリ詳細へ遷移できるようにした。
- ホーム右上の更新ボタンで、manifest/master取得、SHA-256検証、JSON decode、最低限整合性確認を通したマスタ更新を開始できるようにした。
- ごみ分別カテゴリカードの高さ、アイコン領域、テキスト行数を揃えた。
- 通知テスト、pending通知確認、テスト通知キャンセルを開発者用画面へ分離した。

## 3. シンプルUI維持方針

カレンダータブには詳細カード、凡例カード、予定リスト、検索導線を戻していない。ホームも今週リスト中心のままとし、更新状態は1行バーだけで表示する。

## 4. 安全性

マスタ更新は不正な自治体コード、重複ID、存在しないカテゴリ参照を含むマスタを適用しない。通知テストは `dev-test-notification-` prefix を使い、通常通知のidentifierとは分離した。

## 5. 残課題

左右スワイプの実機感、通知到達、通知許可拒否後のiOS設定復帰は実機で確認が必要。
