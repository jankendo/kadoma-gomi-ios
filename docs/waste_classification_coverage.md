# 分別カテゴリ対応範囲

## 1. 対応カテゴリ

| categoryId | 表示名 | 対応状況 |
| --- | --- | --- |
| burnable | 普通ごみ | 公式ページ確認済み |
| plastic_container | プラスチック製容器包装 | 公式ページ確認済み |
| bottles_cans | びん・缶類 | 公式ページ確認済み |
| paper_cloth | 古紙・古布 | 公式ページ確認済み |
| small_items | 小型ごみ | 公式ページ確認済み |
| pet_bottle | ペットボトル | 公式ページ確認済み |
| bulky | 粗大ごみ | 公式ページ/粗大ごみページ確認済み |
| recycle_law | 市で収集不可 | 共通PDFベース、公式確認推奨 |
| hazardous_note | 注意品目 | 公式ページ/共通PDFベース、公式確認推奨 |

## 2. 代表品目

- 普通ごみ: 生ごみ、紙くず、紙おむつ、靴、かばん、30cm以下のプラスチック製品
- プラスチック製容器包装: 食品トレイ、卵パック、レジ袋、プリンカップ、シャンプーボトル
- びん・缶類: 空きびん、飲料缶、缶詰缶、スプレー缶、簡易ガスボンベ
- 古紙・古布: 新聞、段ボール、雑誌、紙袋、チラシ、牛乳パック、古着、タオル
- 小型ごみ: フライパン、なべ、やかん、陶磁器、ガラス、蛍光管、電池、ライター
- ペットボトル: 飲料用ペットボトル、しょうゆボトル、みりんボトル、酒類ボトル
- 粗大ごみ: 布団、家具、自転車、カラーボックス、スーツケース
- 市で収集不可: テレビ、エアコン、冷蔵庫、洗濯機、パソコン、消火器、タイヤ
- 注意品目: スプレー缶、簡易ガスボンベ、電池、ライター、蛍光管、刃物

## 3. 公式確認推奨

`recycle_law` と `hazardous_note` は通常の集積所に出せない、または品目条件で扱いが変わりやすいため、カテゴリ自体を `needs_review` として扱う。

個別品目でも `requiresOfficialCheck=true` のものは詳細画面で公式確認推奨バッジを表示する。

## 4. 今後の拡張

- PDF分別表2ページ目の全品目をOCR + 人手レビューで完全マスタ化する
- 公式LINEの品目検索と照合できる運用フローを検討する
- 素材別、サイズ別、危険物別の補助質問を追加する
