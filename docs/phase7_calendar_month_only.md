# Phase 7 Calendar Month Only

## 1. 方針

カレンダータブは「いつ何を出すか」を見る画面に限定し、月間カレンダー表示を主役にした。詳しい分別方法は検索タブとごみ分別タブに任せる。

## 2. 画面構成

1. 緑ヘッダーの画面タイトル
2. 短い出し方案内
3. 月タイトル、地区名、前月/次月、今日ボタン
4. 必要時のみの1行注意
5. 月間カレンダーグリッド

## 3. 月間グリッド

- 日曜始まりの7列。
- 曜日行は日曜赤、土曜青。
- 日付セルは大きくし、画面高さから行高を計算する。
- 今日は薄い緑背景とアクセント枠で示す。
- 選択日は緑枠で示す。
- ごみ種別は短いラベルまたはアイコンで最大2件まで表示する。
- 小型画面では1件目のアイコン中心にし、残りは `+N` で示す。

## 4. 削除した詳細

選択日詳細、凡例カード、リスト表示、分別手順、通知情報、検索導線は表示しない。カレンダーセルのVoiceOverでは日付と収集予定を読み上げる。

## 5. 年末年始

12月・1月など公式確認が必要な月は、1行注意だけを表示する。未確認の例外日を確定表示しない。
