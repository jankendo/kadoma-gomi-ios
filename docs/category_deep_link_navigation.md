# Category Deep Link Navigation

## 1. 目的

ホームとカレンダーで見つけたごみ種別から、検索を挟まずに該当する分別カテゴリ詳細へ進めるようにする。

## 2. 遷移元

- ホーム: 今週収集日リストのごみ種別ラベル。
- カレンダー: 月間カレンダーセル内のごみ種別ラベル。

## 3. 遷移先

既存の `WasteCategoryDetailView` を使用する。表示内容は分別カテゴリ名、一言説明、出し方、代表品目、注意点、公式情報リンク。

## 4. 例外時の扱い

`categoryId` に対応するカテゴリが存在しない場合はリンクを表示しない。収集なしの日はタップ不可。

## 5. UI方針

タップ可能性は小さな `chevron.right` やカテゴリラベルのリンク化で示す。ホームとカレンダーのシンプルな見た目を壊す大きなボタンや詳細カードは追加しない。

## 6. VoiceOver

カテゴリラベルは「普通ごみ、詳細を開く」のように読み上げる。日付セル全体の読み上げとは分け、カテゴリ詳細への操作であることを明確にする。
