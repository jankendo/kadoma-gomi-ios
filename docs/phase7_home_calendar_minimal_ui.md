# Phase 7 Home / Calendar Minimal UI

## 1. 目的

第7フェーズでは、ホームとカレンダーから補助情報をさらに削り、毎日見る画面として一瞬で読める構成へ絞り込んだ。

## 2. 参考画像から取り入れた思想

- 緑色のシンプルなヘッダー。
- ホームは地区情報と今週の収集日リストを中心にする。
- カレンダーは月間グリッドを画面の主役にする。
- 行間とセルを大きくし、文字を読みやすくする。
- 詳細説明や補助カードを増やさない。

参考画像のアプリ名、キャラクター、固有アイコン、他自治体情報は使用していない。

## 3. ホームで削除・縮小したもの

- 大きな今日カード。
- 分別手順カード。
- クイックアクション群。
- データ状態カード。
- 複数の注意カード。
- 装飾的なカード表現。

ホームに残した主な情報は、地区情報、今日から7日分の収集日リスト、必要時のみの短い注意バー。

## 4. カレンダーで削除・縮小したもの

- 月/リスト表示切替。
- 選択日詳細カード。
- 大きな凡例カード。
- 下部の収集予定リスト。
- 分別方法説明。
- 検索や通知への導線。
- 長い注意文。

カレンダーに残した主な情報は、短い出し方案内、月移動、地区名、月間グリッド、必要時のみの1行注意。

## 5. 維持した機能

- 門真市限定。
- A-F地区対応。
- 大倉町1-20=A地区プリセット。
- 225品目以上の検索辞書。
- 分別カテゴリと出し方データ。
- master生成、schema validation、manifest/SHA。
- 通知予定プレビュー。
- ライトモード専用。

## 6. 残課題

Windows環境では、Xcode Preview、Simulator、実機表示、VoiceOver実走査、Dynamic Type最大の目視確認は未実施。GitHub Actions macOS runnerでビルドを確認する。
