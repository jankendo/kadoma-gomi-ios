# ホーム 今日のごみ UX

## 1. 目的

アプリを開いた瞬間に「今日は何のごみの日か」だけでなく、「それをどう出すか」まで分かるようにする。

## 2. 追加した表示

`HomeTodayWasteGuideCard` を追加し、今日の収集予定を優先して以下を表示する。

- ごみ種別
- 収集日
- 出し方の要約
- 代表例
- 出す前の手順
- 注意点
- 品目検索への導線

今日の収集がない場合は、明日、それもなければ次回予定を表示する。

## 3. 情報の優先順位

1. 今日の収集予定
2. 今日の出し方
3. 明日の収集予定
4. 検索・カレンダー・通知・地区のクイック導線
5. 次回予定
6. 年末年始・公式確認注意

## 4. 小型画面対応

- 代表例は `ViewThatFits` で横並び/縦並びを切り替える
- 出し方ステップは最大4件まで表示
- 複数収集日は最大2カテゴリまで詳細表示し、残り件数を短文で知らせる

## 5. アクセシビリティ

- カード全体を読み上げ可能なまとまりにする
- 色だけに依存せず、アイコン・カテゴリ名・テキストを併用する
- ボタンは大きめの `controlSize(.large)` を使う

## 6. Phase 6 シンプルホーム

- ホームを「今日のごみカード」だけでなく、「直近7日間の収集予定リスト」を主軸に再構成した。
- 今日のごみカードでは、分別種類、カテゴリ説明、最初にやることを短く表示する。
- 参考画像のように日付ごとの予定を縦に並べるが、門真市A-F地区と既存masterの情報を使う独自UIとして実装した。
- 詳細説明は分別タブと検索詳細へ逃がし、ホームは毎朝の確認に集中させた。
