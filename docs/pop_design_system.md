# Pop Design System

## 1. カラーパレット

- 背景: `#E9F8F2`、`#F7FBF3`、`#FFF7E8`
- メイン: `#0A8F8A`
- サブ: `#2F7EEB`
- アクセント: `#F59E0B`
- 注意: `#D97706`
- エラー: `#D94B4B`
- 公式確認: `#54748A`

## 2. ごみ種別色

- 普通ごみ: コーラル
- プラスチック製容器包装: グリーン
- びん・缶類: ブルー
- 古紙・古布: ブラウン
- 小型ごみ: パープル
- ペットボトル: シアン
- 粗大ごみ: ブラウンオレンジ
- 注意品目: コーラル

## 3. タイポグラフィ

画面タイトルと今日/明日カードは太め、説明文は`callout`以上を中心にし、小型画面でも読める階層にしました。

## 4. カード

角丸を大きめにし、白いカードと柔らかい影で押しやすく見える表現にしています。重要カードは背景グラデーションと淡い枠線を使います。

## 5. アイコン

外部画像は追加していません。SF Symbolsを使い、著作権不明素材を排除しました。
