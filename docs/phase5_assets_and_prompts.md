# Phase 5 アセットと画像生成プロンプト

## 1. 今回の方針

著作権不明の画像は使用しない。既存のライトモード専用ポップUIと整合させるため、今回はSF Symbols中心のベクター表現を採用した。

外部画像の無断取得や低品質な大量画像投入は行っていない。

## 2. 追加したアイコン定義

`AppIcon.swift` に以下を追加した。

- `guide`: 分別ガイド
- `steps`: 出し方ステップ
- `examples`: 代表例
- `rinse`: 洗う/すすぐ
- `remove`: 外す
- `bundle`: しばる/まとめる
- `reserve`: 予約
- `separate`: 分ける

## 3. 使用画面

- ホームの出し方ガイドカード
- 分別ガイド一覧
- カテゴリ詳細
- 品目詳細
- 検索結果カード

## 4. 画像生成の有無

Phase 5では画像生成ツールによるビットマップ生成は行っていない。理由は、リポジトリ内で確実に再現できるSF Symbols/SwiftUIベクターの方が、CIビルド・アクセシビリティ・著作権リスクの面で安全なため。

## 5. 将来画像生成する場合のプロンプト

### ホーム用イラスト

門真市の生活ごみカレンダーアプリ向け。明るいライトモード、公共サービスとして信頼できる、子どもっぽすぎない、シンプルなフラットイラスト。ごみ袋、カレンダー、チェックマーク、朝の光を組み合わせる。背景透過、角丸カードに配置しやすい余白、文字なし。

### 分別ガイド用イラスト

日本の自治体ごみ分別アプリ向け。プラスチック容器、空き缶、新聞束、ペットボトル、粗大ごみを柔らかい色で並べたフラットベクター風。アイコンとして使いやすく、線は太め、情報を邪魔しない。文字なし。

### 注意品目用イラスト

スプレー缶、電池、ライター、割れ物を安全に分けるイメージ。怖すぎない注意表現、コーラルとアンバーのアクセント。公共アプリ向け、文字なし、背景透過。
