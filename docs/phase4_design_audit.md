# Phase 4 Design Audit

## 1. 現在の実装概要

門真市限定、A-F地区、大倉町1-20=A地区、master `2026.04.01-af.3`、検索225品目、Dec/Jan `needs_review` ルールを維持したSwiftUIアプリです。第4フェーズではデータ層を変更せず、DesignSystemと主要画面の見た目・操作性を中心に監査しました。

## 2. デザイン上の課題

- 既存配色は公共系として安定していたが、毎日開く生活アプリとしては少し硬い。
- Dark Previewとdark launch colorが残り、ライト専用方針と矛盾していた。
- カレンダーは月表示のセルが小さく、選択日の詳細が弱かった。
- 検索は225品目に拡張済みだが、画面内の検索入口がナビゲーションバーに寄っていた。
- 設定は情報量が増え、ライトモード専用方針や表示の説明が不足していた。

## 3. 維持すべき品質

- 門真市限定方針。
- A-F地区と大倉町1-20プリセット。
- master生成、schema validation、manifest/SHA。
- 検索正規化、危険物・粗大ごみ候補・公式確認推奨。
- 通知予定プレビュー。
- GarbageSummaryProvider。
- GitHub Actions build/pages/source-monitor。

## 4. 第4フェーズ方針

ライトモード専用に固定し、明るい背景、柔らかいカード、読みやすい大きめの見出し、色・アイコン・ラベル併用のごみ種別表現へ刷新します。誤表示につながるデータ推定は行わず、年末年始は引き続き未確認として注意表示します。
