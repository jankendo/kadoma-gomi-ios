# Calendar Swipe Navigation

## 1. 仕様

- 左スワイプ: 翌月へ移動する。
- 右スワイプ: 前月へ移動する。
- 前月/次月ボタンと今日ボタンは維持する。
- 月移動後は `displayedMonth` と `selectedDate` を同じ月の先頭日に更新する。
- A-F地区の収集予定は既存の `store.events(on:)` から再計算される。

## 2. 実装方式

`CollectionCalendarView` の月間グリッドに `DragGesture` を追加した。誤作動を避けるため、横移動量が72ptを超え、かつ縦移動より十分大きい場合だけ月移動する。

## 3. UI制約

カレンダー画面は月間カレンダー表示のみを主役にする。スワイプ対応のために下部詳細カード、予定リスト、凡例カードは追加しない。

## 4. アクセシビリティ

前月/次月/今日ボタンは引き続きタップで操作できる。スワイプ操作が難しいユーザーでも同じ操作が可能。

## 5. 検証観点

- 左スワイプで翌月になる。
- 右スワイプで前月になる。
- 月タイトルと日付グリッドが一致する。
- 地区別収集予定が月移動後も表示される。
- iPhone SE相当でも月間表示が縮みすぎない。
