# Phase 4 Design Upgrade

## 1. 目的

「明るく、楽しく、迷わず、ごみの日が分かる」アプリへ近づけるため、ライトモード専用のポップなUIへ刷新しました。

## 2. 実装範囲

- `UIUserInterfaceStyle=Light` と `.preferredColorScheme(.light)` によるライト固定。
- `AppColor`、`AppTypography`、`AppRadius`、`AppShadow`、`AppIcon` の刷新。
- 背景グラデーション、柔らかいカード、明るいごみ種別色。
- ホーム、カレンダー、検索、設定、オンボーディングのポップ化。
- Dark Previewの削除とLight専用Previewへの置換。

## 3. デザイン方針

- 背景は薄いミントからクリームへ流れる明るい色。
- 主色は親しみやすいティール、補助色はブルー、アクセントはアンバー。
- ごみ種別は色・SF Symbols・短いラベルを併用。
- 注意表示はアンバー/コーラルで見落としにくくしつつ、怖すぎない表現にする。
- カードの角丸と影を広げ、押せる要素を分かりやすくする。

## 4. 画面改善

- ホーム: 今日/明日カード、地区カード、クイックアクションを明るく再構成。
- カレンダー: 月ヘッダー、凡例、日付セル、選択日詳細、年末年始注意カードを再設計。
- 検索: 画面内検索欄、例チップ、結果カード、注意バッジを強化。
- 設定: 地区、通知、データ、表示方針、公式情報をカードで整理。
- オンボーディング: 門真市A-F地区対応と大倉町1-20プリセットを明記。

## 5. 実装しないこと

- 他自治体対応。
- 未確認例外日の確定登録。
- 外部画像素材の導入。
- データ生成スクリプトやmaster構造の破壊的変更。
