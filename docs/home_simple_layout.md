# Home Simple Layout

## 1. 目的

ホームは、アプリを開いてすぐ「今日出すごみ」「今週の予定」「地区」を確認するための実用画面にする。

## 2. 構成

1. 地区情報エリア
2. 今日のごみカード
3. 直近7日間の収集予定
4. よく使う機能
5. 年末年始・公式確認注意
6. データバージョン表示

## 3. 地区情報エリア

`HomeDistrictBand` で、門真市の住所、地区名、公式更新日を表示する。タップすると設定画面へ移動できる。大倉町1-20=A地区プリセットは維持する。

## 4. 今日のごみカード

`TodayCollectionSummaryCard` で、今日の収集予定、カテゴリ名、出し方の要点を表示する。今日収集がない場合は「今日は収集予定なし」と明記し、直近予定とカレンダーへ自然に誘導する。

## 5. 直近7日間

`WeeklyCollectionList` で、今日から7日分を縦リスト表示する。参考画像のように日付を左、収集予定を右に置き、収集なしも文字で表示する。

## 6. クイック導線

検索、分別、カレンダー、地区・通知設定を2列で表示する。装飾は控え、行動名と短い説明を優先する。

## 7. 注意表示

年末年始や公式確認推奨は、強すぎないアンバー背景で表示する。未確認情報は断定しない。

## 8. アクセシビリティ

今日のごみカードと直近予定行は `accessibilityElement(children: .combine)` を使い、日付と収集予定がまとまって読まれるようにする。

## 9. Phase 7: 今週リスト中心化

第7フェーズで、今日カード、クイックアクション、データ状態カードをホームから外した。ホームは、地区情報と今日から7日分の収集日リストを中心にする。

各行は左に `M/d(E)`、右に収集ごみ種別を表示する。収集がない日は `収集なし` を薄い色で表示する。今日の行は薄い緑背景で示す。詳しい分別方法は検索タブとごみ分別タブで確認する。
